Otobi Keyboard

Bangla Bijoy-like layout for PC and android

কম্পিউটারে ইংরেজি লিখতে কি কোনো software ইন্সটল করতে হয়? অবশ্যই না। ইংরেজির মতো বাংলা লিখতেও কোনো third party sofware ইন্সটলের প্রয়োজন নেই। কেবলমাত্র operating system কে জানিয়ে দেয়া প্রয়োজন আপনি কোন লেআউট ব্যবহার করে বাংলা লিখতে চান।
অটবি কিবোর্ড টি Windows অপারেটিং সিস্টেম এর জন্য ডেভেলপ করা হয়েছে। এটি Unicode Standard কে সম্পূর্ণভাবে সমর্থন করে। এর মাধ্যমে Windows এ কোনো third party sofware ছাড়াই সরাসরি বাংলা লেখা যায়। এতে কিবোর্ড লেআউট হিসেবে বিজয় লেআউট ব্যবহার করা হয়েছে। এটি কেবলমাত্র Windows এ বিজয় লেআউট টি সংযোজন করে দেয়। এরপর Windows এর নিজস্ব Input Method System বিজয় লেআউটের মাধ্যমে বাংলা বর্ণ সরাসরি কিবোর্ড থেকে input নিতে পারে।
